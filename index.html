PaperTrade

AI-powered algorithmic trading platform

Live Demo

Full-stack paper trading platform where users create algorithmic strategies in plain English and the app converts them to executable logic via Claude AI. Connects to Alpaca Markets API for real-time data and simulated trade execution.

What I built

  • → Multi-user auth system with JWT, bcrypt, and per-user encrypted API key storage
  • → AI strategy parser that converts natural language into trading parameters using Claude API
  • → Multi-factor signal engine combining RSI, MACD, Bollinger Bands, and market regime detection
  • → Real-time dashboard with Chart.js, Alpine.js reactivity, and HTMX dynamic updates
  • → CI/CD pipeline: GitHub Actions → Docker → Fly.io, with Alembic migrations on deploy

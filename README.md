# b-lesk.github.io

Personal portfolio site for Benjamin Lesko — Data & Analytics Engineer.

Live at [b-lesk.github.io](https://b-lesk.github.io)

## Stack

Single-page static site. No build step, no dependencies.

- HTML + CSS (custom, no frameworks)
- [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk) + [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) via Google Fonts
- Hosted on GitHub Pages

## Local preview

Open `index.html` directly in a browser — no server needed.
